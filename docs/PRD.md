# AI Panel Studio — 产品需求文档 (PRD)

## 1. 产品概述

**AI Panel Studio** 是一个 AI 圆桌讨论 Web 应用。用户输入任意议题，系统动态生成虚拟主持人和多领域专家，进行实时 AI 驱动的圆桌辩论。

## 2. 目标用户

| 用户群体 | 特征描述 | 核心诉求 |
|---------|---------|---------|
| 知识工作者与研究人员 | 需要快速多角度了解议题 | 几分钟内获得不同视角的深度分析 |
| 产品/决策人员 | 希望在讨论或策略会前预演论据 | 发现盲点、验证假设、提升决策质量 |
| 教育与内容创作者 | 需要将复杂观点结构化呈现 | 获得可录制、可回放的"虚拟专家访谈" |

## 3. 功能需求

### 3.1 首页
- 展示所有讨论列表
- 支持新增讨论
- 显示讨论状态（待开始/进行中/已完成等）

### 3.2 嘉宾生成
- 用户输入议题和专家数量
- 系统调用 LLM 动态生成主持人 + 专家库
- 展示姓名、职业、头衔、领域、配色标识

### 3.3 演播厅模式
- 主持人开场发言
- 专家按顺序发言，每次 1-2 句
- 支持暂停/继续
- 实时转录显示

### 3.4 专家状态小窗
- 每个专家独立 UI 卡片
- 实时显示 Agent 运行状态（待命/准备发言/发言中）
- 当前关注点/公开思考摘要
- 不展示隐藏的 chain-of-thought

### 3.5 实时共识与分歧
- 讨论过程中实时更新共识和分歧区域

### 3.6 总结
- 讨论结束后主持人发表自然语言总结
- 不展示 JSON

## 4. 技术需求

- 前后端分离架构
- SQLite 数据库
- SSE 实时更新
- API Key 仅后端环境变量读取
- 多讨论完全隔离
- 中文 UI
- 无前端框架依赖

## 5. 技术选型

| 层级 | 技术 |
|------|------|
| 后端框架 | Python FastAPI |
| 数据库 | SQLite (aiosqlite) |
| 实时通信 | Server-Sent Events (SSE) |
| LLM API | Deepseek API (兼容 OpenAI 格式) |
| 前端 | 纯 HTML + CSS + Vanilla JS |
| 测试 | pytest + pytest-asyncio |
| 项目管理 | uv (Python 包管理) |
